# Make-A-Request
🎵 Concept: QR Song Request App

Goal: Guests scan a QR code → enter their name + song → host/DJ receives the request instantly.

How it works (user flow)

Guest scans QR code at the venue.

A web page opens with a simple form:

Name

Song request

Optional message (e.g., “for Sarah!”)

They tap Submit.

The DJ/host receives the request in real time.

🧩 Core Features
Guest side

QR code access (no app download needed)

Simple form

Confirmation message (“Your request was sent!”)

DJ/Host side

Live dashboard of requests

Mark songs as played

Filter duplicates

Optional voting/upvotes

🔔 How the DJ Gets Notified

Options:

Live dashboard(might change due to updates)
